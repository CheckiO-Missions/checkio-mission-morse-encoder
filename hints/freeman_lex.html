Since MORSE is a dictionary, use its get() method with two parameters. The first is "c" from generator expression, because you want to receive value (combination of characters) for "c" key. The second parameter (default value in case of key absence in dictionary) use whitespace. Its a tricky moment I explain you next step. 

(MORSE.get(c, ' ') for c in text.lower())
I have generator expression, but I need string. Help to get it!

Generator expression is an Iterable and usually serves as a parameter for some function. MORSE keys and values are strings, so you need no additional transformation. You may use string method join() to merge all characters into one string. 

separator.join(MORSE.get(c, ' ') for c in text.lower())
How should I get needed space between words and characters in final string?

You need to have 3 whitespace between words and 1 whitespace between characters in a word. So use 1 whitespace (" ") as separator, it gives you one whitespace between all characters. And here is the trick I promised you. In original text the word separator is 1 whitespace. Since there are no such key in MORSE, it takes default value - 1 whitespace instead. In your final string you use 1 whitespace as separator between all characters. So 1 whitespace separator + 1 whitespace as "character" + again 1 whitespace separator gives you desirable 3 white spaces as a separator between words in final string! 

" ".join(MORSE.get(c, ' ') for c in text.lower())
That is cool! But how the final solution should looks like?

Actually, that almost all. You just need to add function definition: 

def morse_encoder(text: str) -> str:

    return ' '.join(MORSE.get(c, ' ') for c in text.lower())

The hints solution is brilliant, but you can make it even more compact, using lambda expression. 

morse_encoder = lambda text: ' '.join(MORSE.get(c, ' ')
                                      for c in text.lower())

If your previously add whitespace key-value pair into MORSE, you don't need of default whitespace value inside get(). Besides you may use map() function which implement some function to each element of Iterable 

morse_encoder = lambda text: ' '.join(map(MORSE.get, text.lower()))

Bigger one with additional function, splitting the input text by words first, then taking characters from MORSE for each letter in a word, joining them in a new word with 1 whitespace separator, and finally joining all new words with 3 white spaces. 

def morse_encoder(text: str) -> str:

    def trans(word):
        
        return ' '.join(map(lambda char: MORSE[char], word))

    words = text.lower().split()
    text1 = '   '.join(map(trans, words))

    return text1
