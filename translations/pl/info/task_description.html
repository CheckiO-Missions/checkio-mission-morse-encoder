Państwa zadaniem jest zaszyfrowanie wiadomości tekstowej przy użyciu adresu Morse code. Tekst wejściowy będzie składał się z liter (wielkich i małych), cyfr i białych spacji. Nie będzie żadnych znaków specjalnych ('&', '?', '#' itp.)
Proszę używać 3 spacji między słowami i 1 spacji między każdą literą każdego słowa.

Dane wejściowe: Tajna wiadomość (str{% if is_js %}ing{% endif %}).

Wyjście: Ta sama wiadomość (str{% if is_js %}ing{% endif %})ale zaszyfrowana.

Przykłady: 

{{init_code_tmpl}}

Jak jest używany: Do kryptografii i pracy szpiegowskiej.

Warunki wstępne:

  • 0 < len(text) < 50;
  • Tylko angielskie litery (wielkie i małe), cyfry i białe spacje.
